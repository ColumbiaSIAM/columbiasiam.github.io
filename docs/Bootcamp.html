Columbia SIAM Coding Bootcamp

Application

What is it?

The 6th annual Coding Bootcamp will focus on teaching Python to high school students (9th-12th grade) and building analytical toolboxes for a future in STEM. The program is entirely FREE! 

When and where is it?

The 2022 edition of the coding bootcamp will be held virtually over Zoom. There are five sessions:

  • 2/5/22 - Introduction to Python
  • 2/19/22 - Introduction to Calculus
  • 3/5/22 - Linear Algebra
  • 3/26/22 - Probability and Statistics
  • 4/9/22 - Data Science
